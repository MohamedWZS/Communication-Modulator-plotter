# Communication-Modulator-plotter

This is a signal Modulation Plotter Script using python.

a time and frequency domain plot is implemented for each of the following Modulation Techniques:
[1] DSB-LC <br/>
[2] DSB-SC <br/>
[3] SSB <br/>
[4] FM <br/>
[5] PM <br/>
